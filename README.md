# Real Estate Analytics Project

This project uses inquiry data, sales data, and  marketing cost data to analyze project performance, Lead perfoemance, conversion rate, revenue trend, and ROI evaluation for three major real estate developments:

- The Pacific Lagos 
- The Bellagio Lagos 
- Astoria 

The goal is to understand:
- Lead performance  
- Sales conversions  
- Marketing efficiency  
- Revenue trends  
- Project comparison  

All analysis was done using Google Sheets with Pivot Tables and formulas, and summarized in an interactive dashboard.

Key Insights from the Analysis

Inquiry Trends
Bellagio receives the highest volume of inquiries.
Pacific Lagos also recieved volume of Inquiry.
Astoris leads are fewer.

Sales Trends
Bellagio leads in both units sold and revenue.
Pacific Lagos has the best conversion efficiency.
Sales peak in months with stronger marketing investment.

Lead Source Insights
Website and Instagram brought the most leads.

Sales Pipeline Performance
The largest drop-off occurs from *Warm → Hot → Interested*, signaling a need for tighter follow-up.

Marketing ROI
Periods with increased marketing spend directly improve inquiry volume, especially for Pacific Lagos.
